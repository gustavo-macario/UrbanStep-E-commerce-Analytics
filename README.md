# 🚀 SQL Skills: UrbanStep E-commerce Analytics
Este repositório contém a resolução de um desafio técnico de análise de dados para a empresa fictícia UrbanStep, do setor de varejo de moda (streetwear). O objetivo é demonstrar competência em SQL para extração de KPIs, limpeza de dados e geração de insights para tomadores de decisão.

# 📊 Sobre o Projeto
A UrbanStep é um e-commerce que precisava entender melhor seu faturamento, o comportamento regional de seus clientes e a eficiência de seus métodos de pagamento.

O banco de dados simula um ambiente real com 6 tabelas interconectadas:

clientes: Cadastro e localização.

pedidos: Transações e status.

produtos & categorias: Catálogo e margens.

itens_pedido: Detalhamento de SKUs por venda.

pagamentos: Financeiro e métodos de conversão.


# 📈 Desafios Resolvidos
O projeto aborda 10 questões críticas de negócio, incluindo:

Análise de Rentabilidade: Cálculo de lucro bruto por categoria.

Geolocalização: Ticket médio por estado brasileiro.

Financeiro: Taxa de conversão de pagamentos (Boleto vs PIX vs Cartão).

Crescimento: Cálculo de Month-over-Month (MoM).

Estadística: Comparação de Média vs Mediana para identificação de outliers.

Retenção: Tempo médio de ativação de novos clientes.
